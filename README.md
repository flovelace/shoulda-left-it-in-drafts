# Shoulda Left It in Drafts
## Purpose
An API that can be used for social media apps. The main functionality is to create users, allow users to post thoughts, and for other users to react to the thoughts.

## Built with
* JavaScript
* MongoDB
* Mongoose
* Node.js
* Express.js
